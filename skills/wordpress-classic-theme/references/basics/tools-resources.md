# Tools & Resources

**Author:** Damon Cook  
**Published:** September 12, 2022

## Overview

This page presents resources designed to streamline theme development, whether you're creating your first or subsequent WordPress theme. Community-contributed tools supplement the officially documented options.

## Block Themes

Block themes utilize WordPress blocks for construction. Two primary creation tools are highlighted:

### Create Block Theme Plugin

This community-developed plugin enables developers to build standalone or child block themes through these steps:

1. Install the Create Block Theme plugin and latest Gutenberg version
2. Customize site design via the Customizer and Site Editor
3. Refine templates in the Site Editor
4. Access "Create Block Theme" in the WordPress Admin Dashboard under Appearance
5. Download the generated zip file containing all theme files

### Theme Generator

Created by Carolina Nymark, this tool offers four starting options:

- No Code: blank theme for non-developers
- Empty: six templates with basic theme.json
- Basic: includes templates, template parts, and patterns
- Advanced: robust setup with multiple templates and extensive theme.json configuration

## Classic Themes

The **_s (Underscores)** starter theme provides initialization support for PHP-based classic theme development.

## Tools

**WordPress Coding Standards for PHP_CodeSniffer**

A ruleset validating WordPress code quality and adherence to established best practices regarding coding style along with established best practices regarding interoperability, translatability, and security.

**WPThemeReview Standards**

These sniffs automate code analysis during the Theme Review Process using static analysis.

**Theme Check Plugin**

This plugin enables developers to run WordPress.org's automated testing before submission, identifying potential issues early.
