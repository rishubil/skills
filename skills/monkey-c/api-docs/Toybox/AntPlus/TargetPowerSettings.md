## Class: Toybox.AntPlus.TargetPowerSettings

## Overview

Represents the target power set on fitness equipment when the equipment is in target power training mode. Fields may return `null` so you should `null` check values before using them.

## Instance Member Summary

- [**targetPower**](#targetPower-var "targetPower (Var)") as [Lang.Float](https://developer.garmin.com/connect-iq/api-docs/Toybox/Lang/Float.html) or **Null**
