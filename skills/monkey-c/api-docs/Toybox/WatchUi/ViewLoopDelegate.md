## Class: Toybox.WatchUi.ViewLoopDelegate

## Overview

A delegate object for the view loop. The next and previous events generated by the system are passed to the delegate object when [ViewLoop](https://developer.garmin.com/connect-iq/api-docs/Toybox/WatchUi/ViewLoop.html) is active for the app.

## Instance Method Details

### initialize(viewLoop as WatchUi.ViewLoop)

Constructor for the delegate

### onNextView() as Lang.Boolean

Represents the next page behavior.

This is typically triggered by the down button ([KEY\_DOWN](https://developer.garmin.com/connect-iq/api-docs/Toybox/WatchUi.html#KEY_DOWN-const)) or by a [SWIPE\_UP](https://developer.garmin.com/connect-iq/api-docs/Toybox/WatchUi.html#SWIPE_UP-const) [SwipeEvent](https://developer.garmin.com/connect-iq/api-docs/Toybox/WatchUi/SwipeEvent.html) on a touch screen.

### onPreviousView() as Lang.Boolean

Represents the previous page behavior.

This is typically triggered by the up button ([KEY\_UP](https://developer.garmin.com/connect-iq/api-docs/Toybox/WatchUi.html#KEY_UP-const)) or by a [SWIPE\_DOWN](https://developer.garmin.com/connect-iq/api-docs/Toybox/WatchUi.html#SWIPE_DOWN-const) [SwipeEvent](https://developer.garmin.com/connect-iq/api-docs/Toybox/WatchUi/SwipeEvent.html) on a touch screen.
