## Class: Toybox.WatchUi.ViewLoopDelegate

## Overview

A delegate object for the view loop. The next and previous events generated by the system are passed to the delegate object when [ViewLoop](ViewLoop.md) is active for the app.

## Instance Method Summary

- [**initialize**](#initialize-instance_function "initialize (Instance Function)") (viewLoop as [WatchUi.ViewLoop](ViewLoop.md))
	Constructor for the delegate.
- [**onNextView**](#onNextView-instance_function "onNextView (Instance Function)") () as [Lang.Boolean](../Lang/Boolean.md)
	Represents the next page behavior.
- [**onPreviousView**](#onPreviousView-instance_function "onPreviousView (Instance Function)") () as [Lang.Boolean](../Lang/Boolean.md)
	Represents the previous page behavior.

## Instance Method Details

### initialize(viewLoop as WatchUi.ViewLoop)

Constructor for the delegate

### onNextView() as Lang.Boolean

Represents the next page behavior.

This is typically triggered by the down button ([KEY\_DOWN](../WatchUi.md#KEY_DOWN-const)) or by a [SWIPE\_UP](../WatchUi.md#SWIPE_UP-const) [SwipeEvent](SwipeEvent.md) on a touch screen.

### onPreviousView() as Lang.Boolean

Represents the previous page behavior.

This is typically triggered by the up button ([KEY\_UP](../WatchUi.md#KEY_UP-const)) or by a [SWIPE\_DOWN](../WatchUi.md#SWIPE_DOWN-const) [SwipeEvent](SwipeEvent.md) on a touch screen.
