## Overview

Thrown if the [GenericChannel.initialize()](https://developer.garmin.com/connect-iq/api-docs/Toybox/Ant/GenericChannel.html#initialize-instance_function) is unable to acquire an ANT channel from the system. This indicates that all available ANT channels are in use.
