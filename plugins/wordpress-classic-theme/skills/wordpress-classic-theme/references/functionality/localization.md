# Localization

**First published:** October 23, 2014  
**Last updated:** December 13, 2019

## What is localization?

Localization refers to the subsequent process of translating an internationalized theme. The abbreviation `l10n` represents this concept, derived from the ten letters that fall between the initial and final character.

For comprehensive guidance on implementing localization in themes, the WordPress documentation directs developers to consult the dedicated Localization section within the Common APIs handbook.
